Cross Platform Blackjack Game - Windows (WinForms)


This is the Windows GUI version Blackjack game built in C#. The application features a user controled event driven interface.
The GUI updates based on the selections of the player

Getting Started: 
-User will need Windows OS
-Visual Studio 2022 or newer
-.NET Desktop Development Workload installed

Steps to Run:
-Clone repository
https://github.com/winniedel/Black-Jack-GUI-WinForms.git
-Open BlackJack GUI.sln
-Set WinForms as the Startup Project
-Run the program


Players will be able to:
-View their current hand
-Choose to Hit or Stand during each round
-Select a New Game to restart the program
-Track their individual score along with the dealers'


Gameplay:
-The player starts with one card from a shuffled deck, and can either choose to Hit or Stand
-The dealer automatically draws from the same shuffled deck until they reach a total of 17 or higher
-Scores update and iterate by one based off of the round's winner
-Clicking 'New Game' clears the board for a new round
